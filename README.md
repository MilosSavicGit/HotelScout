# HotelScout

**Part of ScoutPlatform**

HotelScout is a browser-based and Android-compatible application for
discovering accommodation **along a driving route** or **within a 3 km
radius** of your current location.

Unlike traditional booking websites that focus only on destinations,
HotelScout helps travellers discover places to stay **during the
journey**.

HotelScout is one of the core applications of **ScoutPlatform**, a
family of lightweight exploration tools built on open geographic data.

------------------------------------------------------------------------

# Overview

HotelScout searches OpenStreetMap in real time to locate accommodation
near a selected route or around the user's current location.

The application displays accommodation on an interactive map and
provides structured information whenever it is available.

# Design Philosophy

HotelScout performs lightweight live searches instead of downloading
large regional accommodation databases.

Workflow:

1.  Select a route or use your current location.
2.  Search for accommodation in real time.
3.  Display accommodation in travel order or by distance.
